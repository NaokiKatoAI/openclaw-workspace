## りんちゃんの大冒険 RPG開発開始 🎮

### 概要
- かっぴーがRPG「りんちゃんの大冒険」を作りたいと依頼（#ゲーム制作チャンネル）
- 要件定義→Phase 1プロトタイプまで一気に進めた

### 決定事項
- **主人公**: りんちゃん（黒柴犬）= かっぴーのペット
- **ジャンル**: ターン制RPG、トップダウン2D、ドット絵
- **プラットフォーム**: ブラウザゲーム（Phaser 3 + TypeScript + Vite）
- **規模**: 全8章、10時間+プレイ
- **エンカウント**: ランダム
- **パーティ**: 3人（りん固定＋仲間2人: 野良猫、フクロウ）
- **素材**: AI生成
- **雰囲気**: ダーク・シリアス

### ストーリー（ネタバレ）
- 「異世界に飛ばされた」と思って冒険するが、実は**100年後のポストアポカリプスの地球**
- 猿の惑星的どんでん返し
- モンスター＝変異した動物/機械、ダンジョン＝廃墟ビル/地下鉄/研究施設
- 伏線: 風化した日本語看板、冷蔵庫、企業ロゴ、動いてるスマホ等
- エンディング3種: TRUE（タイムカプセル発見）、BITTER（廃墟で丸くなる）、HIDDEN（時を戻す）

### 作成ドキュメント
- `rin-rpg/docs/requirements.md` — 要件定義書
- `rin-rpg/docs/enemies.md` — 敵キャラ一覧（91体）
- `rin-rpg/docs/maps.md` — マップ構成（61マップ、町20個）

### Phase 1プロトタイプ完成
- Opus 4.6 sub-agentで開発（約6分半）
- タイトル画面、マップ移動、ターン制戦闘、NPC会話、セーブ/ロード実装
- Cloudflareトンネルで公開（一時URL）

### バグ修正
- 戦闘後スタート地点に戻るバグ → プレイヤー位置引き継ぎ修正
- ガイドダイアログ表示されない → ScrollFactor修正
- WASD操作対応追加
- 町入口に案内テキスト追加

### かっぴーフィードバック
- 「町がひどい」→ 仮素材だから仕方ないが改善必要
- 戦闘の手応えがない → 演出追加済み（フラッシュ、ダメージポップアップ、カメラシェイク等）
- 戦闘後のクリックが面倒 → 自動遷移に変更済み

### 技術メモ

- cloudflared（Cloudflare Tunnel）でローカルサーバーを一時公開
- GitHub Pages等への恒久デプロイはまだ（gh CLIもログイン切れ）

### Phase 2 完了（sub-agent: rin-rpg-phase2）
- **アイテムシステム**: 回復薬、毒消し等7種。インベントリ最大20個
- **装備システム**: 武器/防具/アクセサリー各1枠、9種の装備品
- **ショップシステム**: コノハ（道具屋）、木漏れ日（武器屋・防具屋）、大樹（装備屋）
