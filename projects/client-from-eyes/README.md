# フロムアイズ プロジェクト

## 📁 フォルダ構成

- **CONTEXT.md** - クライアント基本情報（最初に読むファイル）
- **memory/** - 日次メモ（YYYY-MM-DD.md 形式）
- **meetings/** - 議事録
- **resources/** - 資料置き場
  - `contracts/` - 契約書類
  - `brand-assets/` - ロゴ、ブランドガイド
  - `reports/` - 売上レポート、分析資料
  - `images/` - 商品画像、素材
  - `links.md` - 外部リンク集
- **analysis/** - 作業ファイル（キーワード調査、競合分析など）
