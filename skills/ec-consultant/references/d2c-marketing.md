# 独自ドメインEC / D2C マーケティング攻略

## モールとの違い

| 項目 | モール | 自社EC |
|------|--------|--------|
| 集客 | モール任せ | 自力で獲得 |
| 信頼性 | モールの看板 | ブランド構築必要 |
| 顧客データ | 制限あり | 完全に取得可能 |
| 利益率 | 手数料高い | 高利益率可能 |
| ブランディング | 難しい | 自由に構築 |

## 集客チャネル全体像

### 1. ペイド広告（短期・即効性）
- Google広告（検索/ショッピング/P-MAX）
- Meta広告（Instagram/Facebook）
- LINE広告
- TikTok広告
- アフィリエイト

### 2. オーガニック（中長期・資産化）
- SEO / コンテンツマーケ
- SNS運用
- YouTube
- メルマガ / LINE公式

### 3. PR・認知（ブランド構築）
- インフルエンサー施策
- プレスリリース
- メディア掲載
- コラボ企画

## 広告運用

### Google広告

#### 検索広告
- 指名検索は必ず取る
- 競合名キーワードも検討
- 商品カテゴリKWは単価高め

#### ショッピング広告
- 商品フィード最適化必須
- 画像・タイトル・価格が勝負
- P-MAXとの併用推奨

#### P-MAX
- 機械学習で自動最適化
- アセット（画像・動画・テキスト）の質が重要
- シグナル設定でターゲティング誘導

### Meta広告（Instagram/Facebook）

#### クリエイティブが9割
- 最初の3秒で惹きつける
- UGC風が効果的
- 動画 > 静止画（エンゲージメント）

#### オーディエンス設計
- 興味関心ターゲティング
- 類似オーディエンス（LTV高い顧客ベース）
- リターゲティング（訪問者/カート放棄）

#### ファネル設計
1. 認知: 動画広告で興味喚起
2. 検討: カルーセルで商品紹介
3. 獲得: リタゲ + 限定オファー

### TikTok広告
- Z世代・若年層向け
- エンタメ性重視
- Spark Ads（既存投稿を広告化）

### LINE広告
- 日本特化、リーチ広い
- 友だち追加広告でリスト構築
- 年齢層高めにも有効

## SEO / コンテンツマーケ

## ブランディング

### ブランド構築の要素
1. **ミッション・ビジョン**: なぜこのブランドが存在するか
2. **ストーリー**: 創業背景、こだわり
3. **世界観**: ビジュアル、トーン、体験
4. **コミュニティ**: ファンとの関係性

### 差別化ポイント
- 機能的価値（品質、効果）
- 情緒的価値（デザイン、体験）
- 社会的価値（サステナブル、エシカル）

### ブランド体験設計
- 開封体験（パッケージ、同梱物）
- 購入後フォロー
- カスタマーサポート
- コミュニティ参加

## 重要KPI

### 集客
- セッション数
- 新規ユーザー率
- 流入チャネル別構成

### 転換
- CVR（転換率）: 2-3%が目安
- カート投入率
- カート放棄率

### 収益
- 客単価（AOV）
- LTV（顧客生涯価値）
- 粗利率

### 広告効率
- CPA（顧客獲得単価）
- ROAS（広告費用対効果）
- CAC（顧客獲得コスト）

### リピート
- リピート率
- 購入頻度
- 解約率（サブスク）

## 成功の方程式

```
売上 = セッション × CVR × 客単価
利益 = 売上 × 粗利率 − 固定費 − 広告費
```

### LTV > CAC × 3 を目指す
- CAC回収期間は3ヶ月以内が理想
- LTVを上げる施策に投資
- CAC高騰時はオーガニック強化

## フェーズ別戦略

### 立ち上げ期（0→1）
- PMF（プロダクトマーケットフィット）検証
- 少額広告でテスト
- 熱狂的ファン10人作る

### 成長期（1→10）
- 勝ちパターンに広告投下
- UGC・口コミ加速
- LTV向上施策

### 拡大期（10→100）
- 新規チャネル開拓
- 商品ライン拡充
- オフライン展開も検討
