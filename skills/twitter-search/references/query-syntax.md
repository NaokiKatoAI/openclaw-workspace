# Twitter検索クエリ構文

## 基本演算子

### キーワード検索
```
ECサイト運営
```
AND検索（すべてのキーワードを含む）

### OR検索
```
ECサイト OR ネットショップ
```

### 完全一致
```
"楽天市場での売上アップ"
```

### 除外
```
EC -Amazon
```
"EC"を含むが"Amazon"を含まない

## ユーザー指定

### 特定ユーザーのポスト
```
from:username
```

### 特定ユーザーへのリプライ
```
to:username
```

### 特定ユーザーへの言及
```
@username
```

## ハッシュタグ・URL

### ハッシュタグ
```
#楽天市場
```

### URLを含む
```
url:amazon.co.jp
```

## エンゲージメント

### 最小リツイート数
```
ECサイト min_retweets:10
```

### 最小いいね数
```
ECサイト min_faves:50
```

### 最小返信数
```
ECサイト min_replies:5
```

## 言語・場所

### 言語指定
```
ECサイト lang:ja
```

### 場所指定
```
ECサイト place:東京
```

## 複雑なクエリ例

### ECコンサル関連で人気のポスト
```
(ECコンサル OR EC運営) min_retweets:10 lang:ja
```

### 特定ユーザーの楽天関連ポスト
```
from:competitor_account 楽天
```

### Amazon以外のECサイト情報
```
ECサイト (楽天 OR Yahoo OR Shopify) -Amazon
```

## 制限事項

- 検索対象は**過去7日間**のみ（無料・Basicプラン）
- リアルタイム検索ではない（数秒の遅延あり）
- 1リクエストあたり最大100件
