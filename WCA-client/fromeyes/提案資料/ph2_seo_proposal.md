**株式会社WCA**

## 📌 ご提案の概要

Ph2リニューアルに合わせて、**Google検索結果の表示改善**と**AI検索への対応**を行うことで、クリック率・流入数の向上を目指します。

---

## 🔍 施策①: 商品ページの検索結果を強化する

### 【現状】Before
Google検索で「リフレア ワンデー」と検索した場合：

```
ワンデーリフレアUVモイスチャー38 30枚入り× 1箱 | リフレア公式通販
清潔で安心、瞳にやさしい1日使い捨てコンタクトレンズ、ワンデーリフレア
UVモイスチャー38のご紹介。メーカー直販だから可能になったお手頃価格...
```
→ タイトルと説明文だけ。他のサイトと見た目が同じ。

### 【改善後】After

```
ワンデーリフレアUVモイスチャー38 30枚入り× 1箱 | リフレア公式通販
★★★★★ 4.82（レビュー件数）  ¥1,406  在庫あり
清潔で安心、瞳にやさしい1日使い捨てコンタクトレンズ...
```
→ **星評価・価格・在庫状況**が表示される。目立つのでクリック率が上がる。

**実装内容**: Product + AggregateRating 構造化データの追加

## 🔍 施策②: FAQ（よくある質問）を検索結果に展開表示する

### 【現状】Before
「リフレア 定期 解約」で検索しても、FAQページが通常の検索結果として表示されるだけ。

```
よくある質問 | リフレア公式通販
コンタクトレンズに関する様々な疑問にお答えするQ&A・よくある質問の
コーナーです...
```

### 【改善後】After

```
よくある質問 | リフレア公式通販
コンタクトレンズに関する様々な疑問にお答えするQ&A...

▼ 定期便は解約できますか？
  はい、マイページからいつでも解約可能です。次回お届け日の
  7日前までにお手続きください。

▼ 送料はかかりますか？
  ネコポスなら全国送料無料です。宅配便の場合は825円（税込）...
```
→ **Q&Aがそのまま検索結果に展開表示**される。ユーザーはクリックしなくても回答が見える。

**さらに**: ChatGPTやGeminiなどのAIが「リフレアの定期便は解約できる？」と聞かれた時に、このQ&Aを引用して回答する可能性が高まります。

**実装内容**: FAQPage 構造化データの追加 + FAQ内容のテキスト化

## 🔍 施策④: 画像→テキスト化によるSEO改善

### 【現状の課題】
現在のサイトは、ナビゲーション・ボタン・セクションタイトル等が**ほぼ全て画像**で作られています。

人間には見えていますが、**Googleは画像の中の文字を読めません**。

例：
- ヘッダーメニュー「1DAY」「2WEEK」→ 画像（SVG/PNG）
- 「使用期間で選ぶ」「シリーズで選ぶ」→ 画像
- FAQページのカテゴリボタン → 画像

### 【改善後】
これらをHTML テキスト + CSS デザインに変更することで：
- Googleがサイトの内容を正しく理解できるようになる
- 検索順位の改善が期待できる
- ページの表示速度も向上する（画像の読み込みが減る）

**実装内容**: 画像テキストのHTML化（デザインはCSSで維持）

## 🔍 施策⑤: AI検索への対応（note運用）

### 【背景】
ChatGPT、Gemini、Perplexity等のAI検索が普及し、「ゼロクリック検索」（検索結果をクリックせずにAIの回答で完結）が増加しています。

### 【課題】
現状、AIが「おすすめのコンタクトレンズは？」「シリコーンコンタクトの選び方は？」と聞かれても、refrearの情報を引用する根拠となるコンテンツがありません。

### 【対策】
noteブログで**第三者目線の記事**を発信：
- 「シリコーンコンタクトレンズの選び方」
- 「1DAYと2WEEKどっちがいい？メリット・デメリット比較」
- 「コンタクトレンズの定期購入はお得？」

AIがこれらの記事を参照し、回答の中でrefrearを推薦する可能性が高まります。

**実装内容**: note記事の定期投稿（月2〜4本目安）
