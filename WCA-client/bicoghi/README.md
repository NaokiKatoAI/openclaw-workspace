### 企業理念・ビジョン
- **創業**: 1967年
- **社名の由来**: BICO = 美光（美しく光り輝くこと）
- **経営指針**: 「誠実・謙虚・努力」
- **ミッション**: 人と人をつないで信頼を築きあげること
- **哲学**:
  - 「あたりまえ」のレベルアップ
  - 互いに「信頼」できる絆をつくる
  - 思いやりとバランス
