## 商品特性の整理

- **価格帯**: ¥9,900〜¥13,200 → TikTok Shopの衝動買いレンジにギリ入る

- **ターゲット**: ペットオーナー（特にシニアペット飼い主、ペットの健康意識が高い層）

## 🎯 TikTok動画戦略（3月スタート想定）

### Phase 1: 認知拡大（3月〜）— まず「見てもらう」

#### ① ビフォーアフター動画（最優先）
- **内容**: マットに乗せる前の落ち着きないペット → マットの上でぐっすり眠る姿
- **狙い**: 効果の可視化、驚き → シェア・保存されやすい
- **構成例**: 
  - 冒頭2秒「うちの子、夜鳴きが止まらなくて…」（テキスト表示）
  - マット投入 → ペットがくつろぐ映像
  - 最後にショップタグ
- **尺**: 15〜30秒

#### ② もふもふ癒し動画
- **内容**: マットの上で気持ちよさそうに寝てるペットをひたすら映す
- **狙い**: TikTokのペットカテゴリは再生数が伸びやすい。商品認知を自然に
- **構成例**: 癒しBGM + ペットのアップ + さりげなくマットにロゴ
- **尺**: 15〜60秒

#### ③ 「プラウシオン®って何？」解説動画
- **内容**: 特殊繊維の仕組みを簡単に解説（図解 + ペット映像）
- **狙い**: 信頼性・科学的根拠のアピール
- **構成例**: 「ナノレベルの鉱物加工で血流改善」→ 学会発表のエビデンス → 実際のペット映像
- **尺**: 30〜45秒

### Phase 2: 購買促進（4月〜）— 「買いたい」にする

#### ④ 飼い主のリアルレビュー動画
- **内容**: 実際に使っている飼い主さんのリアルな声
- **狙い**: UGC風コンテンツは信頼性が高い
- **ポイント**: 「29歳の猫がピタッと泣き止んだ」などのエピソードは超強力

#### ⑤ シニアペット特化コンテンツ
- **内容**: 「シニア犬/猫の飼い主さんへ」と絞ったメッセージ
- **狙い**: ターゲット明確化、感情に刺さる
- **構成例**: 「15歳の柴犬、最近足が…」→ マットで血流改善 → 元気になった姿

#### ⑥ 開封動画
- **内容**: 届いた箱を開ける → マットを広げる → ペットの反応
- **狙い**: TikTokの定番フォーマット、購買後のイメージを湧かせる

### Phase 3: ライブコマース（5月〜体制が整い次第）

#### ⑦ ライブ配信
- **内容**: ペットと一緒にマットの魅力を紹介
- **狙い**: リアルタイムの質疑応答 → 購買ハードル下げ
- **ポイント**: 
  - ライブ限定クーポン配布
  - ペットが実際にくつろいでる様子をリアルタイムで見せる
  - 視聴者のペットの悩みに答えるQ&Aコーナー

## 📱 コンテンツ制作のポイント

### TikTokで伸びるペット動画の法則
1. **冒頭2秒で掴む** — テキストフックまたはペットの意外な行動
2. **音楽はトレンドを使う** — TikTokのトレンド音源を常にチェック
3. **縦型（9:16）必須**
4. **テキスト字幕を入れる** — ミュートで見てる人も多い
5. **ハッシュタグ**: #ペット #犬のいる暮らし #猫のいる暮らし #シニア犬 #ペットの健康 #TikTokShop #circule

### 生成AI活用（提案時の想定通り）
- ナレーション生成（TTS）
- サムネイル・テキストデザイン
- BGM選定サポート
- 動画の構成案出し
