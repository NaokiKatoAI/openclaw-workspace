# ヨークス EC戦略設計（仮）

**策定日**: 2026年2月9日  
**前提条件**: 
- 月額コンサル費: 25〜30万円
- 契約期間: 1年間（2027年1月〜12月想定）
- 目標売上: 6,000〜6,500万円（現状5,500万円）
- 広告費: 別途（コンサル費とは別）

---

## エグゼクティブサマリー

### 現状認識
- **強み**: 楽天4.9の高評価、ライセンスブランド（ANNA SUI等）、オーダーメイド手袋
- **弱み**: レビュー数の少なさ、Amazon展開の不足、通年商材の不足、モール依存
- **機会**: Amazon展開、コンテンツSEO、通年商材開発、越境EC
- **脅威**: 競合の強さ（グローブデポ20,000件超レビュー）、季節性

### 戦略の基本方針
**"高品質・専門性・ブランド力"** を軸にした差別化戦略
- 競合（グローブデポ）の物量勝負ではなく、質と専門性で勝負
- ライセンスブランドとオーダーメイドを2本柱に
- モール依存からの脱却（自社EC比率を現状の〜%から30%へ）
- 通年販売体制の構築（季節依存からの脱却）

---

## SWOT分析（総合）

### Strengths（強み）
1. **楽天4.9の高評価** → 商品品質・サービスの高さの証明
2. **ライセンスブランド** → ANNA SUI等の有名ブランド
3. **オーダーメイド手袋** → 競合にない独自サービス
4. **専門競技用手袋** → フェンシング、アメフト等のニッチ市場
5. **東かがわ市の産地** → 手袋の聖地というブランド
6. **実店舗の存在** → O2O施策の可能性

### Weaknesses（弱み）
1. **レビュー数の少なさ** → 競合比で圧倒的に少ない（最大の弱点）
2. **Amazon展開の不足** → 現状1商品のみ
3. **通年商材の不足** → 冬場依存からの脱却が課題
4. **モール依存** → 楽天依存度が高い（自社ECの弱さ）
5. **SNS活用の不足** → Instagram等の運用が弱い
6. **コンテンツ不足** → ブログ、読み物コンテンツの不足

### Opportunities（機会）
1. **Amazon市場** → 未開拓の巨大市場
2. **コンテンツマーケティング** → 手袋の専門知識を活かしたSEO
3. **通年商材開発** → UV対策、運転用、スポーツ用等
4. **越境EC** → 海外での日本製手袋需要（特に米国、欧州）
5. **サブスク型サービス** → 季節ごとのお届けサービス
6. **SNSマーケティング** → Instagram、TikTok等でのビジュアル訴求

### Threats（脅威）
1. **競合の強さ** → グローブデポの圧倒的なレビュー数（20,000件超）
2. **季節性** → 冬場メインの季節商材
3. **ファストファッション** → 低価格競争
4. **モール手数料の上昇** → 利益率の圧迫
5. **既存取引先の問題** → オックス社のスケジュール遅延等

---

## 戦略の3本柱

### 柱1: モール最適化（短期収益確保）
**目的**: 既存の楽天・Yahoo!を徹底最適化し、短期的な売上を確保

### 柱2: 自社EC強化（中長期的な利益確保）
**目的**: モール依存からの脱却、利益率の高い自社EC比率を向上

### 柱3: 新規チャネル開拓（成長戦略）
**目的**: Amazon、越境EC等の新規チャネルで成長を加速

---

**楽天・Yahoo!・自社EC共通**

##### 施策A: 購入後フォローメール
- **内容**: 商品到着1週間後にレビュー依頼メール自動送信
- **特典**: レビュー投稿で次回使える500円クーポン進呈
- **KPI**: 月間レビュー獲得数30件以上

##### 施策B: 梱包時のカード同梱
- **内容**: 「レビューをお願いします」カードを全商品に同梱
- **デザイン**: QRコード付き、手書き風メッセージ
- **コスト**: カード印刷費 月間¥10,000程度

##### 施策C: レビューキャンペーン
- **内容**: 月間レビュー投稿者の中から抽選でギフト券プレゼント
- **予算**: 月間¥20,000（¥5,000 × 4名）
- **告知**: メルマガ、SNS、商品ページ

##### 期待効果
- 3ヶ月で100件以上のレビュー獲得
- ショップ評価4.9の維持
- CVR 0.3〜0.5%向上

#### 1-2. 楽天RPP広告最適化
**現状の広告運用の見直しと最適化**

- **入札単価**: 競合分析に基づく最適化

##### KPI
- **ROAS**: 300%以上

- **売上**: 広告経由で月間¥500,000以上

- オーダーメイドの通年受注

- SNSでの有機的な拡散

- 広告運用の最適化
