# ユーザーフロー

**作成日**: 2026-02-06

---

## 初回訪問ユーザー

1. **ランディングページ**
   - サービス説明
   - サンプル画像（Before/After）
   - 「今すぐ始める」ボタン

2. **ログイン/登録画面**
   - メールアドレス登録
   - ソーシャルログイン（Google/Apple推奨）
   - Supabase認証

3. **プラン選択画面**
   - 無料プラン（3枚）を選択
   - または有料プラン選択 → Stripe決済

4. **メインアプリ画面へ**

---

## メインアプリ画面（加工フロー）

1. **画像アップロード**
   - ドラッグ&ドロップ
   - ファイル選択ボタン
   - 対応形式：JPEG/PNG/WebP

2. **プレビュー表示**
   - アップロードした元画像を表示

3. **リサイズ選択**
   - プルダウンメニュー
     - Instagram正方形（1080x1080）
     - Instagram縦長（1080x1350）
     - Twitter（1200x675）
     - Facebook（1200x630）
     - YouTube サムネイル（1280x720）
     - オリジナルサイズ

4. **「フィルター適用」ボタン**
   - クリックで昭和映画フィルター適用
   - 処理中インジケーター表示
   - 加工後画像をプレビュー表示
   - **クレジット消費なし**（何度でも試せる）

5. **「ダウンロード」ボタン**
   - クリックでクレジット-1
   - JPEG形式でダウンロード
   - スマホ：写真フォルダへ自動保存
   - PC：通常のダウンロード

6. **クレジット残数表示**
   - 画面上部に常時表示
   - 「残り○枚」

---

## クレジット切れ時

1. **ダウンロードボタンをクリック**
   - 「クレジットが不足しています」モーダル表示

2. **プラン変更画面へ誘導**
   - ライトプラン（500円/30枚）
   - プロプラン（1,000円/無制限）
   - Stripe決済

3. **購入完了 → メインアプリ画面へ**

---

## 既存ユーザー（再訪問）

1. **ランディングページ**
   - 「ログイン」ボタン

2. **ログイン画面**
   - Supabase認証

3. **メインアプリ画面へ**
   - 前回のプラン・クレジット残数を引き継ぎ
