東京アプリ→dポイント交換 完全ガイド
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 結果：11,000pt → 12,100pt（+1,100pt）
■ 期限：2026年3月31日 23:59まで（+10%増量）


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP1: 準備するもの
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【必要なもの】
1. スマホ（NFC対応必須）
   - iPhone: iOS 16以降
   - Android: Android 11以降

2. マイナンバーカード（有効期限内）
   - 利用者証明用パスワード（4桁）
   - 券面事項入力補助用パスワード（4桁）
   ⚠️ 忘れたら市区町村窓口で再設定が必要

3. アプリ2つ
   - 「東京アプリ」
   - 「デジタル認証アプリ」


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP2: アプリをダウンロード（2つ）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【2-1. 東京アプリ】
1. App Store（iPhone）or Google Play（Android）を開く
2. 検索バーに「東京アプリ」と入力
3. 「東京都公式アプリ」を見つけてダウンロード
4. 「開く」を押さずに待機

【2-2. デジタル認証アプリ】
1. 同じくストアで「デジタル認証」と検索
2. 「デジタル認証アプリ」をダウンロード
3. 「開く」を押さずに待機


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP3: 東京アプリで登録
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【3-1. 東京アプリを開く】
1. ホーム画面から「東京アプリ」をタップ
2. 「新規登録」ボタンをタップ
3. 利用規約を読んで「同意する」
4. メールアドレス・パスワードを入力
5. 「登録」ボタンをタップ
6. 届いたメールのリンクをタップして認証

【3-2. プロフィール設定】
1. 東京アプリに戻る
2. ログイン（さっき設定したメールアドレス＋パスワード）
3. 生年月日・性別などの基本情報を入力
4. 「保存」をタップ


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP4: デジタル認証アプリで登録
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【4-1. デジタル認証アプリを開く】
1. ホーム画面から「デジタル認証アプリ」をタップ
2. 「利用登録」をタップ
3. 利用規約を読んで「同意する」
4. アプリのパスワード（6桁）を設定
5. 「次へ」をタップ


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP5: マイナンバーカードで本人確認（重要！）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️【事前準備】
- Wi-Fiを切る（設定 → Wi-Fi → オフ）
- マイナンバーカードを手元に準備
- 2つの暗証番号を確認：
  - 利用者証明用パスワード（4桁）
  - 券面事項入力補助用パスワード（4桁）

【5-1. 東京アプリに戻る】
1. 東京アプリを開く
2. ホーム画面に「東京ポイント申込み」ボタンがあるのでタップ
   ※なければ「メニュー」→「東京ポイント」をタップ
3. 「マイナンバーカードで本人確認」ボタンをタップ

【5-2. 本人確認スタート】
1. 「本人確認を開始する」をタップ
2. デジタル認証アプリが自動で開く
3. 「マイナンバーカードを読み取る」をタップ

【5-3. マイナンバーカードをスマホにかざす】
1. 画面の指示に従ってカードをスマホの背面にかざす
   - iPhone: 上部
   - Android: 中央〜下部（機種による）
2. 「利用者証明用パスワード（4桁）」を入力
3. カードを動かさずに5秒待つ
4. 「読み取り成功」と表示されたらOK

【5-4. 券面情報の入力】
1. 「券面事項入力補助用パスワード（4桁）」を入力
2. 再度カードをかざす
3. 氏名・住所が自動入力される
4. 内容を確認して「次へ」をタップ

【5-5. 本人確認完了】
1. 「本人確認が完了しました」と表示される
2. 「東京ポイントを申し込む」ボタンをタップ
3. 「申込み完了」画面が出る


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP6: ポイント付与を待つ（数日〜1週間）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【確認方法】
1. 東京アプリを開く
2. ホーム画面の「ポイント」をタップ
3. 「11,000pt」が表示されたら付与完了


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP7: dポイントクラブアプリの準備
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【7-1. アプリをダウンロード】
1. App Store or Google Playで「dポイントクラブ」と検索
2. 「dポイントクラブ」アプリをダウンロード
3. アプリを開く

【7-2. dアカウント作成】
1. 「dアカウントを発行する」をタップ
2. メールアドレスを入力
3. 届いたメールのリンクをタップ
4. パスワード・生年月日・性別を入力
5. 「登録」をタップ

【7-3. 利用者情報登録】
1. dポイントクラブアプリにログイン
2. ホーム画面の「登録情報」をタップ
3. 「利用者情報登録」をタップ
4. 氏名・住所・電話番号を入力
5. 「登録」をタップ


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP8: キャンペーンエントリー
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【8-1. キャンペーンページにアクセス】
1. dポイントクラブアプリを開く
2. ホーム画面の「キャンペーン」をタップ
3. 「東京ポイント10%増量」を探してタップ
   ※または検索バーで「東京ポイント」と検索

【8-2. エントリー】
1. キャンペーン詳細ページの「エントリー」ボタンをタップ
2. 「エントリー完了」と表示される


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP9: 東京ポイント→dポイント交換
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【9-1. 東京アプリで交換】
1. 東京アプリを開く
2. ホーム画面の「ポイント交換」をタップ
3. 交換先一覧から「dポイント」を選択
4. 交換ポイント数を入力（11,000）
5. 「交換する」をタップ

【9-2. dアカウント連携】
1. 「dアカウントでログイン」画面が表示される
2. さっき作ったdアカウント（メールアドレス＋パスワード）でログイン
3. 「連携を許可する」をタップ
4. 「交換申請完了」と表示される
